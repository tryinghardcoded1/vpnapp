# CV CREATION VPN App

A high-performance VPN application built with Flutter.

## Features
- **Secure Authentication**: Simulated login system.
- **Server Selection**: Dropdown list of global servers.
- **Speed Test**: Visual connection speed analysis.
- **Premium Subscription**: Toggle between Free and Premium plans.
- **Profile Management**: Manage user details locally.

## Getting Started

1. **Install Dependencies**
   ```bash
   flutter pub get
   ```

2. **Run the App**
   ```bash
   flutter run
   ```

3. **Build APK**
   ```bash
   flutter build apk --release
   ```

## Folder Structure
- `lib/`: Main source code
- `assets/`: Images and data files
